# Customization

The game is designed to be easily customizable for different use cases or client needs.

## Configurable Parameters

- Ship speed and rotation sensitivity
- Asteroid count and size
- Projectile speed and lifespan
- Invincibility duration and blink rate

## Visual Adjustments

- Colors and styles can be modified in the rendering layer
- Canvas size and scaling are easily adjustable

## Extensibility Ideas

- Add sound effects and background music
- Implement power-ups or enemy AI
- Introduce levels or progression systems
- Add touch/mobile controls
